## 🖋️ Overview
This repository serves as a digital archive for **CA2124**, a foundational course in Data Structures and Algorithms. The code within demonstrates a transition from theoretical concepts to practical, low-level implementations in C. It covers the essential building blocks of software engineering, focusing on manual memory management, pointer manipulation, and efficient data organization.

---

## 📂 Project Structure

### 🔗 [Linked Lists](./linkedList/)
The foundation of dynamic data storage. This module explores various implementations of the linked list pattern.
- **`linkedlistStr.c`**: Implementation of string-based nodes.
- **`Menu-Normal-linkedList.c`**: A robust, menu-driven interface for standard list operations.
- **`Menu-Ordered-Linked-List.c`**: Logic for maintaining sorted order during insertion.

### 📚 [Stacks & Postfix](./playin_with_stacks/)
Deep dives into the Last-In-First-Out (LIFO) principle and its applications in mathematical expression parsing.
- **`stacks.c`**: Core stack operations (Push, Pop, Peek).
- **`postfix.c`**: A practical application for evaluating postfix mathematical expressions using stack-based logic.

### 🎟️ [Queues](./Queues/)
Exploring the First-In-First-Out (FIFO) principle through both linear and circular structures.
- **`circularQueue.c`**: Efficient use of memory through circular buffering logic.
- **`queues.c` & `PatientQueue.c`**: Real-world simulations, such as managing a patient queueing system.

### 🌳 [Binary Search Trees (BST)](./Tree/)
Moving beyond linear structures into hierarchical data representation.
- **`BST.c`**: A complete implementation of a Binary Search Tree, featuring recursive insertion and traversal algorithms.
